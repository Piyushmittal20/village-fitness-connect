# Village Fitness Connect

A rural fitness platform connecting trainers and users across India, built with multi-language support and easy plan-sharing tools.

## Features

- **Multi-language support** — Switch between English, Hindi, Marathi, Tamil, and Gujarati. All content updates instantly.
- **Workout & diet plan uploads** — Trainers and users can create and save workout plans and diet plans with timestamps.
- **Trainer profiles** — Trainers can showcase their bio, Instagram, and YouTube links via profile cards with clickable social icons.
